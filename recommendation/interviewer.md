# 推荐系统要点

搜广推区别：

|            | 搜索    | 推荐       | 广告    |
|------------|-------|----------|-------|
| 信息送达方式     | 拉     | 推和拉      | 推     |
| 关注点        | 内容消费方 | 内容生成方消费方 | 内容生产方 |
| 是否期待惊喜     | 否     | 是        | 否口    |
| 是否需要集体智慧   | 可能    | 可能       | 需要    |
| 是否需要 query | 需要    | 可能       | 可能    |
| 是否依赖上下文    | 可能    | 可能       | 可能    |
